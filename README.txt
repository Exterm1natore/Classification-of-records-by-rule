Программа: ClassificationOfRecordByRule.

Назначение: Программа предназначена для классификации набора записей по указанному набору правил.

Каждая запись имеет название и набор свойств. 
Каждое свойство имеет своё название и набор целочисленных значений, которые описывают характеристику свойства.

Правила классификации имеют название (класс) и условие(я), при котором запись попадает в данный класс. 
Правила вводятся строго в определённом формате.

Пример записи: Шкаф: цвет=[1,55,2], размер=[2, 11, 99, 64], форма=[15].
Пример свойcтв:
1) Запись принадлежит классу "С покрытием", если у нее есть свойство "покрытие";
2) Запись принадлежит классу "П л о с к и й", если у нее есть свойство "размер", которое представлено двумя значениями;
3) Запись принадлежит классу "Синий", если у нее есть свойство "цвет", в составе которого есть значение "1";
4) Запись принадлежит классу "Матовый", если у нее есть свойство "покрытие" и значение этого свойства равно "[44, 21]".
P.S. Здесь представлены все возможные типы свойств (без дополнтельно ограничения; с количеством целочисленных значений, представленном
в виде текста; в виде единственного целочисленного значения, которое встречается в свойстве записи; с массивом целочисленных значений).

Шкаф: цвет=[1,2]
  ^______________название записи
	^________название свойства записи
             ^___целочисленные значения, которые относятся к определённому свойству

Основная программа хранится по пути (начиная с директории 'Classification-of-records-by-rule'):
.\CourseworkOnKINPO\ClassificationOfRecordsByRule (Windows)

Запускаемый файл '.exe' находится по пути (начиная с директории 'Classification-of-records-by-rule'):
.\CourseworkOnKINPO\ClassificationOfRecordsByRule\release\ClassificationOfRecordsByRule.exe (Windows)

Для работы программы необходимо:
1) запустить командную строку
2) перейти в директорию с исполняемым файлом ('ClassificationOfRecordsByRule.exe')
3) Записать в командную строку следующие аргументы в заданном порядке:
   ClassificationOfRecordsByRule.exe <Путь к файлу с записями> <Путь к файлу с правилами классификации> <Путь к выходному файлу результата>

Общее содержание файлов (начиная с директории 'Classification-of-records-by-rule'):
В директории 'CourseworkOnKINPO' содержатся проекты тестов и главная программа, а так же папка с файлами для проектов тестов.
В директории 'Autotests' содержится файл '.bat' который запускает проект автотестирования. Так же там находятся файлы '.txt' для тестов (входные и выходные)
В директории 'ManualTesting' содержится файл описания ручного тестирования, а так же примеры входных данных.
В директории 'Autodocs' содержатся файлы автодокументации основного проекта.
